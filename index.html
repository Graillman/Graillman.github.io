Suivi Prestation, conçu pour l'équipe Pénélope

Sécurité
Accès réservé à l'équipe L'application est protégée par un code d'accès connu des seuls responsables. Sans lui, impossible de voir ou modifier quoi que ce soit.
Code jamais stocké en clair Le code est transformé en empreinte cryptographique (SHA-256 avec sel aléatoire). Même en cas de fuite de la base, il resterait illisible.
Protection anti-intrusion Après 8 codes erronés, l'adresse qui essaie est bloquée 15 minutes. Deviner le code par force brute est impraticable.
Base de données verrouillée Aucun accès public, sous aucune forme : seul le serveur de l'application peut lire ou écrire (verrouillage RLS). Audit de sécurité Supabase : aucune alerte.
Chiffrement de bout en bout Toutes les communications passent en HTTPS (TLS). Rien ne circule en clair sur le réseau.
Hébergé en France Les données sont stockées à Paris (Union européenne), sur une infrastructure certifiée.
Données minimales Seuls le prénom, le nom, l'accueil et les passages sont enregistrés, rien d'autre. Usage strictement interne ; export et suppression possibles à tout moment.

Bonnes pratiques

Partagez le code uniquement aux responsables concernés et changez-le régulièrement (Réglages → « Changer le code »), par exemple à chaque départ dans l'équipe.
