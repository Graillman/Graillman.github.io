# Suivi Tenues · Pénélope

Application de suivi quotidien des tenues et de l'accueil, pour l'équipe
Pénélope (site Evergreen).

- **Application** : https://graillman.github.io (protégée par code d'accès)
- Cette page ne contient **aucune donnée ni secret** : toutes les données et
  l'authentification vivent côté serveur (Supabase, hébergé à Paris, base
  verrouillée, code haché SHA-256, anti-force-brute).
- Code source complet et documentation : dépôt `copieur-de-jeu`.
